# LISTNERS - What's Inside This App

## Features You Get

### Video Calling
- Call teacher directly on video
- See incoming call on screen
- Accept or reject call

### Ask Questions
- Post a question
- Teachers answer you
- Mark correct answer

### Chat
- Chat with teacher
- Send images
- Real-time messages

### Courses
- Watch video courses
- Enroll in courses
- Buy courses with eSewa

### Quiz
- Play quizzes
- Test your knowledge

### Wallet
- Earn points for answering
- Withdraw to eSewa

### Admin Panel
- See all users
- See all payments
- Manage everything
